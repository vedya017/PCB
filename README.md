Arduino-Based Gas Sensor Detection System with Custom PCB Design

Designed and developed a custom PCB integrated with an Arduino Uno-based gas monitoring system for the detection of hazardous sewage gases, including hydrogen sulfide (H₂S), methane (CH₄), ammonia (NH₃), and carbon monoxide (CO). The system continuously acquires gas concentration data from multiple gas sensors, processes the readings through the Arduino controller, and compares them against predefined safety thresholds. When gas levels exceed permissible limits, the system generates immediate alerts through output indicators to warn workers of potentially dangerous conditions. Real-time gas concentration values are displayed on an LCD screen, LED lights, and buzzers for appropriate warning levels, enabling continuous environmental monitoring and improving safety in sewage and confined-space operations.

Project Flow:
Gas Sensors → Signal Acquisition → Arduino Uno Processing → Threshold Comparison → LCD Display of Gas Concentrations → Hazard Detection → Alert/Warning Generation for Workers.
